For purposes of early development purposes, a simple media type has
been defined for testing integration between client and controller code.
This media type is JSON based, and is not to be used in any production code.
This media type does not fully support all data that can be within a given DFD,
only providing one level.

Definition for a DFD (note there are no nested DFDs in this definition)
{
    "id": "string",
    "label": "string",
    "type": "string",
    "originator": "string",
    "genericType": "string",
    "subDFDNode": "string",
    "nodeList": 
    [
        {
            "id": "string",
            "type": "string",
            "label": "string",
            "x": "value",
            "y": "value"
        }
    ],
    "linkList": 
        [
            {
                "id": "string",
                "type": "string",
                "label": "string",
                "origin_id": "string",
                "dest_id": "string",
                "x": "value",
                "y": "value"
            }
        ],
    "subDFDNodeList": 
    [
        {
            "id": "string",
            "type": "string",
            "label": "string",
            "x": "value",
            "y": "value"
        }
    ]
}

Definition for a single node
{
    "id": "string",
    "type": "string",
    "genericType": "string",
    "label": "string",
    "x": "value",
    "y": "value",
    "originator": "string",
    "linkList": 
    [ 
        { 
            "id": "string",
            "label": "string"
        }
    ]
}

Definition for a single subDFDNode
{
    "id": "string",
    "type": "string",
    "genericType": "string",
    "label": "string",
    "x": "value",
    "y": "value",
    "originator": "string",
    "subDFD": "string",
    "linkList": 
    [ 
        { 
            "id": "string",
            "label": "string"
        }
    ]
}

Definition for a single link
{
    "id": "string",
    "type": "string",
    "genericType": "string",
    "origin_id": 
        {
            "id": "string",
            "label": "string"
        },
    "dest_id": 
        {
            "id": "string",
            "label": "string"
        },
    "originator": "string",
    "x": "value",
    "y": "value"
}
